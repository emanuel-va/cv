# Curriculum Vitae

Responsive Curriculum Vitae built with HTML & CSS, available in both light & dark themes.

## Documents

- [light cv](documents/cv-light.pdf)
- [dark cv](documents/cv-dark.pdf)

## Preview

![light theme](assets/img/mockups/light.png)

![dark theme](assets/img/mockups/dark.png)

## Features

- ATS-friendly PDF
- Text-selectable PDF
- Print-ready A4 format
- Updatable blueprint
- Light & Dark themes
- Responsive layout
- Hosted on GitHub
- Version controlled with Git

## Technologies

- HTML5
- CSS3
