## Features

- ATS-friendly PDF
- Text-selectable PDF
- Print-ready A4 format
- Updatable blueprint
- Light & Dark themes
- Responsive layout
- Hosted on GitHub
- Version controlled with Git

## Technologies

- HTML5
- CSS3
